# Slutetpåkön.se

En hemsida för att se slutet på den kilometerlånga kön till lundakarnevalsuppropet baserat på crowdsourcad data.

<img src="assets/3.png" style="width: 30%;">

## Funktioner

- **Realtidsposition:** Använde crowdsourcad data för att visa ett ungefärligt slut på kön.
- **Crowdsourcing:** Använde användarens platsinformation för att dela data om köns slut.

## Teknik

Vi deployade på en digitalocean VPS med nginx.

- **Frontend:** React, Tailwind, Leaflet för karta
- **Backend:** Node, Sqlite
